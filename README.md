# git basic user guide
## init workcopy environment
- mkdir [DIR]
- cd [DIR]
- git init
- git clone [URL]
## workcopy to stage
- add/rm
- checkout
## stage to HEAD
- commit
- reset
## HEAD to remote reposition
- push
- pull
## get specified version file
- git log [FILE]
- git checkout [version] [FILE]
## reset to specified version file
- git log [FILE]
- git checkout [version] [FILE]
- git add
- git commit
